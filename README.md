Encryptor/Decryptor text with password
(LINK)[https://kucheriavyiy.github.io/crypto/]

Crypto) If you have any text you want to protect, enter it in the upper left field. Set a password below and click Crypto.

Decrypto) If you have some text you want to decrypt, enter it in the right field. Set a password in the lower left and click DeCrypto.
